# 剪切板管理器实现

## 摘要
为 Clipper Tauri 应用程序实现一个精简的剪切板管理器，支持文本和图片内容预览、可配置的存储限制、搜索功能，以及跨 Windows 和 macOS 的系统托盘集成。

## 变更概述
此变更引入专注的剪切板管理功能，具有未来内容类型的扩展性，使用策略模式进行内容处理。

## 范围
- **包含功能**: 文本和图片剪切板监控、图片预览、可配置存储限制、搜索功能、系统托盘集成、启动配置
- **不包含功能**: 云同步、文件/复杂格式支持、加密、数据导出、大文本监控、分类标签

## 技术方案
- 使用策略模式的 Rust 后端进行内容类型处理
- React 前端采用最简化 UI 设计
- SQLite 持久化存储，带大小限制
- 平台特定的剪切板 API（文本+图片）
- 系统托盘集成用于后台运行

## 用户影响
用户将能够：
1. 自动跟踪复制的文本和图片内容
2. 在历史记录中查看图片缩略图和文本预览
3. 搜索剪切板历史记录
4. 配置存储限制和个人偏好
5. 通过全局快捷键和系统托盘访问剪切板
6. 控制启动行为

## 依赖项
- 用于剪切板访问的额外 Rust crates
- 剪切板监控可能需要的系统权限

## 成功标准
- 成功监控剪切板变化
- 存储和检索历史项目
- 历史选择的响应式 UI
- 跨平台键盘快捷键处理
